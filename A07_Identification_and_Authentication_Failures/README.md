# A07 – 身份识别与认证失败（基于弱 JWT 认证的账户接管攻击复现）

---

## 项目简介

本项目复现 **OWASP Top 10 2021 – A07：Identification and Authentication Failures（身份识别与认证失败）** 漏洞类别，并以**红队攻击链**的方式进行完整实践。

项目通过构建一个存在缺陷的 JWT 认证系统，展示攻击者如何从身份认证弱点出发，最终实现**管理员账户接管**。

---

## 项目目标

* 复现真实世界中的认证机制设计缺陷
* 深入理解 JWT 身份认证机制及其安全风险
* 构建完整攻击链
* 展示攻击与防御双视角
* 提供安全修复与检测思路

---

## 漏洞类别

OWASP Top 10（2021）

**A07 – 身份识别与认证失败**

当系统在用户身份验证、会话管理或认证逻辑设计中存在缺陷时，攻击者可能绕过认证或冒充合法用户。

本项目涉及的认证失败包括：

* 弱 JWT 密钥设计
* 过度信任客户端 Token
* Token 无失效机制
* 缺少认证安全加固措施

## 漏洞应用设计（故意存在安全缺陷）

本 Web 应用采用 JWT 进行身份认证，但存在多处安全设计问题：

| 模块         | 设计情况      |
| ---------- | --------- |
| 身份认证方式     | JWT Token |
| 签名算法       | HS256     |
| 密钥管理       | 环境变量弱密钥   |
| 会话管理       | 无状态       |
| Token 失效机制 | 未实现     |
| 登录保护       | 无爆破限制   |

---

## 攻击场景

攻击者通过分析认证机制，利用 JWT 配置缺陷伪造身份，实现管理员权限获取

### 攻击链流程

1. 认证机制信息收集
2. JWT Token 结构分析
3. 弱密钥发现与利用
4. 伪造管理员 Token
5. 权限提升
6. 管理员账户接管

---

## 攻击影响

* 身份认证绕过
* 权限提升
* 未授权数据访问
* 持续身份冒充
